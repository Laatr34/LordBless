Ce que nous avons appris en développement web

HTML

HyperText Markup Language

Comme nous l’avons appris durant notre devoir, le HTML est considéré comme le squelette de notre site web, car il constitue tous les éléments que celui-ci doit contenir.

CSS

Cascading StyleSheets

Comme le HTML est le squelette, le CSS est le langage qui forme la chair et les vêtements stylistiques de notre site web.

JS

JavaScript

Si le HTML est le squelette et le CSS l’apparence, JavaScript est le cerveau du site web. Il permet d’ajouter de l’interactivité et de rendre la page dynamique.
